# Cognitive Governance - PhD Synopsis Presentation

A minimalist, Jony Ive-inspired presentation for the PhD synopsis on "Cognitive Governance: The Role of Executive Decision Hygiene in Modulating AI-Augmented Strategy Under Uncertainty"

## Overview

This presentation explores how executive decision hygiene practices can improve strategic decision-making when using AI-augmented forecasts in early-stage ventures.

**Candidate:** Syed Faraaz
**Institution:** Nirwan University Jaipur
**Subject:** Management (PhD)

## Research Synopsis

### Key Topics

1. **Introduction** - AI-augmented decision making in startups
2. **Research Gap** - Bridging behavioral strategy and algorithmic advice
3. **Objectives** - Understanding decision hygiene practices
4. **Hypotheses** - Testing structured decision processes
5. **Methodology** - Three-phase mixed-method approach
6. **Significance** - Academic and practical contributions

### Core Concept

The research investigates how **decision hygiene** practices (independent pre-judgement, structured discussions, explicit dissent) can help founders and leadership teams make better strategic decisions when using AI forecasts, while maintaining psychological safety and voice within teams.
